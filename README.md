# VTS - Virtual Typing Simulator

## Overview

VTS is a modern, futuristic-styled typing practice application designed to help users improve their typing skills through structured exercises. The application features a sleek dark theme with glowing effects and provides real-time feedback on typing accuracy and speed.

## Features

- **Futuristic UI Design**: Dark gradient backgrounds with glowing cyan accents and modern typography
- **Comprehensive Typing Exercises**: 27 different exercises covering all keyboard sections
- **Real-Time Feedback**: Live WPM (Words Per Minute) calculation as you type
- **Accuracy Tracking**: Visual highlighting of correct and incorrect characters
- **Error-Adjusted WPM**: Calculates adjusted typing speed accounting for errors
- **Responsive Design**: Adapts to different screen sizes
- **No External Dependencies**: Pure HTML, CSS, and JavaScript implementation

## Exercise Categories

1. **Introduction** - Overview of typing fundamentals
2. **Home Row Basics** - ASDF JKL; keys practice
3. **Home Row Expansion** - G and H keys integration
4. **QWERT Practice** - Top row left side keys
5. **YUIOP Patterns** - Top row right side keys
6. **ZXCVB Drills** - Bottom row left side keys
7. **NM,./ Exercises** - Bottom row right side keys
8. **Number Basics** - Numeric key practice
9. **Symbol Practice** - Special characters and symbols
10. **Capitalization Practice** - Shift key exercises
11. **Numeric Exercises** - Number pad and calculations

## How to Use

1. **Select an Exercise**: Use the dropdown menu to choose a typing exercise
2. **Start Typing**: Click in the text area and begin typing the displayed text
3. **Monitor Progress**: Watch your WPM update in real-time
4. **View Results**: Click "Done!" to see your final WPM and error count
5. **Reset**: Use the "Reset" button to start an exercise over

## Keyboard Shortcuts

While the application focuses on typing practice, you can use:
- **Tab**: Navigate between interactive elements
- **Enter**: Submit when focused on buttons
- **Space**: Normal typing practice
